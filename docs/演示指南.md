# 演示指南

## 1. 准备环境

```bash
npm install
cp .env.example .env
npm run db:init
npm run db:demo
npm run dev:server
npm run dev:web
npm run dev:admin
```

访问地址：

```text
前台：http://127.0.0.1:5173
后台：http://127.0.0.1:5174
API：http://127.0.0.1:4173/api
```

后台管理员：

```text
username: admin
password: admin123456
```

演示用户请以 `server/db/seed-demo.js` 中的固定手机号/账号为准。

## 2. 3 分钟演示流程

### 第 1 分钟：前台整体体验

1. 打开首页。
2. 展示 Hero、精选咖啡、精选图书、社区统计。
3. 切换 Cursor 样式，展示多主题 Cursor。
4. 滚动页面展示 BackToTop、图片优先加载和动画效果。

### 第 2 分钟：用户业务闭环

1. 登录演示用户。
2. 进入图书详情，收藏图书、预约图书、查看评论。
3. 进入商品详情，加入购物车，展示评论点赞/回复。
4. 进入预约页，选择座位、日期、固定时间段、填写信息并确认。
5. 进入积分中心，展示签到、成长值、优惠券兑换。

### 第 3 分钟：后台运营管理

1. 使用 admin 登录后台。
2. 展示 Dashboard 真实统计。
3. 进入商品/图书/活动管理，展示多选、全选、批量操作、图片上传和活动报名名单。
4. 进入座位管理，拖拽座位并说明前台同步。
5. 进入社区审核，展示批量通过/驳回/隐藏和用户处罚联动。
6. 进入操作日志，展示审计记录。

## 3. 5 分钟演示流程

1. 首页浏览：说明 Coffee Book 定位。
2. 登录注册：展示两种登录模式与图形验证码。
3. 商品商城：商品详情、购物车、下单、模拟支付。
4. 图书中心：收藏、预约、评论。
5. 活动中心：活动列表、活动详情、报名；后台可查看报名名单、筛选、导出和更新参加/缺席状态。
6. 社区：帖子详情、评论、回复、点赞；后台处罚后前台展示友好风控提示。
7. 预约：统一座位地图、固定时间段、分阶段确认。
8. 会员：签到、积分、成长等级、优惠券。
9. 后台：Dashboard、批量管理、图片上传、活动报名名单、座位拖拽、社区审核。
10. 数据库与测试：说明真实 DB、演示数据脚本、smoke 测试和 security smoke。

## 7. 常见问题与处理

### 页面空白或进入安全模式
先检查终端和浏览器控制台错误，再执行：

```bash
npm run build
npm run smoke:web
npm run smoke:api
node scripts/security-smoke.js
```

### 数据不够丰富
执行：

```bash
npm run db:demo
```

该脚本幂等，可重复运行。

### 数据库连接失败
检查 `.env` 中 `DB_HOST`、`DB_PORT`、`DB_NAME`、`DB_USER`、`DB_PASSWORD`，并确认 MySQL 已启动。

### 上传图片失败
确认后端启动，`server/public/uploads` 可写，后台登录 token 有效。后台商品/图书/活动上传允许 `user_id = NULL`。

### 安全 smoke 失败
优先查看失败项名称；脚本会自动创建并清理 `security-*` 测试数据。常见原因是 API 服务未启动、数据库未迁移或本地端口被占用。
